# Kullanıcı Kılavuzu - Bölüm 1

## Nasıl Kayıt Olunur?

1. Ana sayfada **"Kayıt Ol"** butonuna tıklayın.
2. Kayıt formunu doldurun:
   - **E-posta:** Geçerli bir e-posta adresi girin
   - **Şifre:** En az 8 karakter olmalı
   - **Rol:** Öğrenci veya Öğretim Üyesi seçin
   - **Öğrenci iseniz:** Öğrenci numarası ve bölüm bilgisi
   - **Öğretim Üyesi iseniz:** Personel numarası, unvan ve bölüm bilgisi
3. **"Kayıt Ol"** butonuna tıklayın.
4. E-posta kutunuzu kontrol edin. Doğrulama linkine tıklayın.
5. E-posta doğrulandıktan sonra giriş yapabilirsiniz.

## Nasıl Giriş Yapılır?

1. Ana sayfada **"Giriş Yap"** butonuna tıklayın.
2. Giriş formunu doldurun:
   - **E-posta:** Kayıtlı e-posta adresiniz
   - **Şifre:** Şifreniz
3. **"Giriş Yap"** butonuna tıklayın.
4. Başarılı giriş sonrası Dashboard sayfasına yönlendirilirsiniz.

**Not:** E-posta doğrulanmamış hesaplarla giriş yapılamaz.

## Profil Nasıl Güncellenir?

1. Giriş yaptıktan sonra sağ üst köşedeki **profil menüsüne** tıklayın.
2. **"Profilim"** seçeneğini seçin.
3. Güncellemek istediğiniz bilgileri düzenleyin:
   - Telefon numarası
   - Adres
   - Hakkımda (Bio)
   - Bölüm bilgisi (öğrenci/öğretim üyesi için)
4. **"Kaydet"** butonuna tıklayın.
5. Başarı mesajı görüntülenecektir.

### Profil Fotoğrafı Yükleme

1. Profil sayfasında **"Fotoğraf Yükle"** butonuna tıklayın.
2. Bilgisayarınızdan bir resim seçin.
3. Yükleme tamamlandığında profil fotoğrafınız güncellenecektir.

### Şifre Değiştirme

1. Profil sayfasında **"Şifre Değiştir"** bölümüne gidin.
2. Mevcut şifrenizi girin.
3. Yeni şifrenizi girin (en az 6 karakter).
4. Yeni şifrenizi tekrar girin (doğrulama).
5. **"Şifreyi Güncelle"** butonuna tıklayın.
